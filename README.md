# 🌐 Personal Website — Krishna Kumar Dey

A premium, modern personal portfolio website built with vanilla HTML, CSS, and JavaScript.

## 🚀 Live Demo

**[https://personal-website-6e2te6xz0-deykris777s-projects.vercel.app/](https://personal-website-6e2te6xz0-deykris777s-projects.vercel.app/)**

## ✨ Features

- **Sleek Dark Mode** — Premium glassmorphic design with a curated color palette
- **Custom Cursor** — Smooth trailing cursor with interactive hover effects
- **Scroll Animations** — Intersection Observer-powered reveal animations
- **Responsive Design** — Fully optimized for desktop, tablet, and mobile
- **Micro-Animations** — Hover effects, floating blobs, and smooth transitions

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid, Backdrop Filter)
- Vanilla JavaScript (Intersection Observer, CSS Animations)
- Google Fonts (Inter, Space Grotesk)
- Font Awesome Icons

## 📂 Project Structure

```
├── index.html    # Main HTML structure
├── style.css     # All styling and animations
├── script.js     # Custom cursor, scroll reveals, nav logic
└── README.md
```
